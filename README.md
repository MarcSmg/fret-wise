# Fretwise - Guitar chord tool

Status: "In progress"

## Description

A guitar tool that displays guitar chord diagrams. Users choose their chords (the root note, chord color, alterations etc.), then the app displays chord diagrams with some of the possible shapes along the guitar neck and the inversions.

Who is it for ?
	Guitarists

## Core features

- Search a chord with the UI
- Display chord diagram
- Display inversions

## When is it done ?

The project succeeds when a user can search a chord and see the chord diagram displayed with some of the possible inversions on the guitar neck.

## Stack

- **React** (MVP)
- **Django + Django REST Framework (DRF)**
